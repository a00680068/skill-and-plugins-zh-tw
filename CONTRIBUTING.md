# 貢獻指南

感謝協助改善繁體中文對照資料。

## 可貢獻項目

- 修正中文名稱或說明。
- 補充官方來源網址。
- 更新已變更的平台功能。
- 回報分類錯誤。
- 改善台灣繁體中文用詞。
- 補充 Claude 模式適配建議。

## 翻譯修改格式

請提供：

1. 英文原名
2. 目前中文內容
3. 建議中文內容
4. 修改理由
5. 官方來源網址
6. 來源檢查日期
7. 校對者名稱
8. 校對狀態

## 翻譯原則

- 保留產品與品牌正式名稱。
- 優先使用台灣常用繁體中文。
- 技術術語可保留英文，但首次出現應補中文說明。
- 不自行擴大工具能力。
- 無法確認時標記「待查證」，不要猜測。

## 模式適配原則

- Claude Chat：對話、研究、分析、內容迭代。
- Claude Code：程式碼庫、終端、測試、除錯、部署。
- Claude Cowork：本機檔案、跨應用程式、連結器與多步驟交付。

模式標記是使用建議，不是排他限制。

若調整模式建議，請同時說明：

- 首選模式
- 可搭配模式
- 信心等級
- 判斷理由
- 是否已人工覆核
