# Security Notes

This project is a static HTML reference tool.

## Security posture

- No backend server is required.
- No database is required.
- No login is required.
- No cookies are intentionally set.
- No API keys, tokens, or secrets are included.
- No external CDN is required.
- No remote JavaScript dependency is loaded.
- The page can run offline.

## User data

The page does not intentionally collect, store, or transmit user data.

The copy-to-clipboard feature only copies prepared prompt text when the user clicks a copy button.

## Reporting issues

If you find a security concern, please open an issue in the public repository after the project is published, or contact the maintainer through the repository contact method.
