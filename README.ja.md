# Skill & Plugins｜スキル & コネクター

これは `Skill & Plugins` の日本語版です。AIスキル、プラグイン型スキル、コネクターを検索するための公開安全版ディレクトリです。

## 使い方

- `index.ja.html` を開くと日本語版を閲覧できます。
- 言語切り替えから `index.html`、`index.zh-CN.html`、`index.en.html` に切り替えられます。
- データファイルは `data/skills-and-connectors.ja.json` と `data/skills-and-connectors.ja.csv` です。

## 収録内容

- 578件のスキル
- 547件のプラグイン型スキル
- 72件のコネクター
- 74件の機能グループ
- 合計650件の対照データ

## 品質状態

日本語版は繁体字中国語の基準版から生成した機械翻訳ベースラインです。人手レビュー待ちの状態を保持しており、最終的な編集済みローカライズとして扱わないでください。
