# Skill & Plugins｜技能与连接器

这是 `Skill & Plugins` 的简体中文版本，用来查询 AI 技能、外挂型技能与连接器。

## 使用方式

- 打开 `index.zh-CN.html` 浏览简体中文版本。
- 可切换至 `index.html`、`index.en.html`、`index.ja.html`。
- 数据文件位于 `data/skills-and-connectors.zh-CN.json` 与 `data/skills-and-connectors.zh-CN.csv`。

## 目前收录

- 578 个技能
- 其中 547 个标记为外挂型技能
- 72 个连接器
- 74 个功能群组
- 共 650 笔中英文对照资料

## 质量状态

简体中文版本由繁体中文基准版经 OpenCC 转换产生，属于可读基线版本，尚未完成逐项人工审校。
