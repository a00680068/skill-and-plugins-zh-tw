# Skill & Plugins | Skills & Connectors

This is the English version of `Skill & Plugins`, a public-safe directory for browsing AI skills, plugin-type skills, and connectors.

## Usage

- Open `index.en.html` for the English version.
- Switch to `index.html`, `index.zh-CN.html`, or `index.ja.html` from the language switcher.
- Data files are available at `data/skills-and-connectors.en.json` and `data/skills-and-connectors.en.csv`.

## Current Coverage

- 578 skills
- 547 plugin-type skills
- 72 connectors
- 74 functional groups
- 650 bilingual records in total

## Quality Status

The English version is a machine-translation baseline generated from the Traditional Chinese source. It keeps a pending human-review status and should not be treated as a final editorial localization.
